# Github Copilotに渡されるプロンプト等の詳細情報

## 特定のツール等がないとシステムプロンプトの`toolUseInstructions`セクションの内容が変わる
- ファイル編集ツールがない場合
- ターミナル実行ツールがない場合
- WEB検索のツールがない場合（この場合は単に該当文章が削除される）
```markdown
You don't currently have any tools available for editing files. If the user asks you to edit a file, you can ask the user to enable editing tools or print a codeblock with the suggested changes.
You don't currently have any tools available for running terminal commands. If the user asks you to run a terminal command, you can ask the user to enable terminal tools or print a codeblock with the suggested command.
Tools can be disabled by the user. You may see tools used previously in the conversation that are not currently available. Be careful to only use the tools that are currently available to you.
```

## カスタムエージェントを使う場合
システムプロンプトの`modeInstructions`セクション内で、エージェントファイル内の指示情報等が追加される。

## Askモードを使う場合は以下のような点がエージェントと異なる
ログから確認できる使えるツールが`search_workspace_symbols`のみになる。これはエージェントのツールにはない、シンボル検索を行うツールである。
（おそらく組み込みでいくつか使えるツールが存在はしているはず）

applyPatchInstructionsセクションやnotebookInstructionsセクションなど、ツールに紐づいたセクションがシステムプロンプトから削除される。

システムプロンプトに`learning_resources\github_copilot\Askモードシステムプロンプト追加.md`の内容が追加される。
- コード検索の方法および、パネルへのMarkdown出力方法の説明が追加される。

## Editもードを使う場合は以下のような点がエージェントと異なる
ログで使えるツールが確認できない。
（おそらく組み込みでいくつか使えるツールが存在はしているはず）

システムプロンプトに`learning_resources\github_copilot\EDITモードのシステムプロンプト.md`の内容が追加される。

## Planモードを使う場合
Planモードは単なる事前定義されたカスタムエージェントである。
それ以外に、特別な挙動は確認できていない。
`C:\Users\user\.vscode\extensions\github.copilot-chat-0.35.2\assets\agents\Plan.agent.md`などからエージェントの内容を確認できる。

## 事前定義されたSavePrompt
これも単なる事前定義されたカスタムプロンプトである。
それ以外に、特別な挙動は確認できていない。
`C:\Users\user\.vscode\extensions\github.copilot-chat-0.35.2\assets\prompts\SavePrompt.prompt.md`などから内容を確認できる。
