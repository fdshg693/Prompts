## 作成手順の概要
以下の手順で、Claudeの公式ドキュメントからプロンプトエンジニアリングに関する箇所を抜き出し、日本語で分かりやすくまとめたドキュメントを作成しました。

### エージェントの作成
1. `ai_coder\github_copilot\agent\web.searcher.md`と同様のエージェントを`.github\agents\web.searcher.agent.md`として作成
2. 作成したエージェントに対して以下のように指示を出す
    - 今回用いた公式ドキュメントのURL`https://platform.claude.com/docs/en/build-with-claude/prompt-engineering`

### エージェントへの指示内容
1. URL配下のドキュメントを読み込んで要約するためのタスク分割を`docs\claude公式\plan\claude_prompting_作業手順書.md`として作成させる
2. 作成された作業手順書を一つずつ実行させる
    - 基本的には、「ステップ○○を行って」 という指示で十分（ステップ3だけ生成ファイルの文字数が多い為、サブエージェントを使って適切にタスク委譲するよう指示を追加）
    - 1および2のステップ1~4は全て別々のセッションで行ったため、計5回の指示で作成された（コンテキスト汚染、トークン数超過を防ぐため）
